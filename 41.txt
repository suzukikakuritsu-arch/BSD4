BSD予想：23, 29, 31, 109 周辺の「素直さ」検証コード
このコードは、判別式の素因数構成（nf）と、局所的な誤差項 a_p の分布（ratio）を計算し、ランクとの相関を可視化する準備を整えるものです。
import numpy as np
import matplotlib.pyplot as plt

def get_ap(A, B, p):
    """楕円曲線 y^2 = x^3 + Ax + B の a_p を計算"""
    if (4 * A**3 + 27 * B**2) % p == 0:
        return 0  # 悪い素数
    
    count = 0
    for x in range(p):
        rhs = (x**3 + A*x + B) % p
        # 平方剰余の判定
        if rhs == 0:
            count += 1
        elif pow(rhs, (p - 1) // 2, p) == 1:
            count += 2
    return p - count

def verify_bsd_logic(A, B, p_limit=100):
    """
    4, 5, 6 の構造と a_p の分布（ratio）を検証
    """
    disc = 4 * A**3 + 27 * B**2
    aps = []
    primes = [p for p in range(2, p_limit) if all(p % i != 0 for i in range(2, int(p**0.5) + 1))]
    
    for p in primes:
        aps.append(get_ap(A, B, p))
    
    # ratio の計算: (p+1-N)/sqrt(p) の平均的な振る舞い
    # ここでは簡易的に a_p の平均的な偏りをスコア化
    ratio = np.mean([abs(a) for a in aps]) / np.sqrt(np.mean(primes))
    
    return {
        "disc": disc,
        "ratio": ratio,
        "aps": aps
    }

# 検証ターゲット（2と3のイタズラが生じるケース）
curves = [
    {"name": "Rank 0 (Stable)", "A": 1, "B": 1},   # 素直なケース
    {"name": "Bridge 23 (Refractory)", "A": -1, "B": 1}, # 23が絡む
    {"name": "Bridge 109 (Complex)", "A": 4, "B": 27} # 109が絡む
]

print(f"{'Curve Type':<20} | {'Disc':<10} | {'Ratio (ap bias)':<15}")
print("-" * 50)

for c in curves:
    res = verify_bsd_logic(c["A"], c["B"])
    print(f"{c['name']:<20} | {res['disc']:<10} | {res['ratio']:.4f}")

# 4,5,6 の「素直さ」を可視化するプロット
plt.figure(figsize=(10, 5))
for c in curves:
    res = verify_bsd_logic(c["A"], c["B"], p_limit=200)
    plt.plot(np.cumsum(res['aps']), label=c['name'])

plt.title("Cumulative sum of a_p (Growth correlates to Rank)")
plt.xlabel("Primes (count)")
plt.ylabel("Sum of a_p")
plt.legend()
plt.grid(True)
plt.show()

    return variance, pressure

2.	混じりけのある系（Ratio が 1.0 に向かう状態）: 5, 7, 23, 29... といった素数が加わることで、系に「雑音（1のイタズラ）」が入り、統計的な比率が平均化（ランダム化）されていく。

この論考における Ratio とは、「数学的な構造の『秩序（規則性）』と『混沌（ランダム性）』の割合」 を数値化したもの、あるいは 「a_p という数値が素数 p に対してどれだけ『素直に』あるいは『ランダムに』振る舞っているかの指標」 であると言えます。

この式における N_p は、**「有限体 \mathbb{F}_p（素数 p で割った余りの世界）における、楕円曲線の解の個数」**を指しています。
少し噛み砕いて説明します。
1. N_p の正体
楕円曲線の式（例えば y^2 = x^3 + Ax + B）に対して、変数 x と y に 0 から p-1 までの数字を当てはめて計算したとき、等号が成り立つペア (x, y) が何組あるか、というカウント数です。（※これに「無限遠点」という特別な点を1つ加えたものが正確な N_p です）。

2. p+1 との関係（なぜ引き算するのか）
もし楕円曲線が完全に「ランダムな式」であれば、解の個数 N_p はだいたい p+1 個くらいになると期待されます。

この a_p は、いわば**「理想的な平均（p+1）からのズレ」**を表しています。
• a_p が正なら、解が平均より少なかった。
• a_p が負なら、解が平均より多かった。
3. 論考における「Ratio」との繋がり
提示された論考では、この「ズレ（a_p）」がどう分布するかを問題にしています。
• 素直な（ランクが低い）曲線: a_p の現れ方に強い規則性（偏り）があります。例えば「ある数値に集中する」ような状態です。
• 反抗期（複雑な）曲線: 新しい素数が混じると、この a_p がバラバラに散らばり、統計的に「ランダムな分布（佐藤・テイト分布など）」に近づきます。
この**「規則的な分布からランダムな分布へ、どれくらい近づいたか」の度合い**を、あなたの論考では Ratio という言葉で表現していると読み解けます。
つまり、N（正確には N_p）は、その曲線がその素数の世界でどれだけ「解（答え）」を持っていたかという実測値であり、a_p はその「個性の強さ（ズレ）」を測るためのスコアのようなものです。
「1のイタズラ」が介入する前の「素直な曲線（CM曲線）」と、介入した後の「反抗期（一般の曲線）」で、a_p の分布がどう変わるかを視覚化するコードを作成します。
このコードでは、以下の2つを比較します：
1.	素直な系: y^2 = x^3 + x （CM曲線、ランク0）→ a_p が特定の規則性を持って分布します。
2.	反抗期: y^2 = x^3 + Ax + B （一般的な曲線）→ a_p がハッセの境界線（|a_p| \le 2\sqrt{p}）の中でランダムに散らばります。

import numpy as np
import matplotlib.pyplot as plt

def get_ap(A, B, p):
    """素数pにおけるa_p = p + 1 - N_p を計算"""
    n_p = 1  # 無限遠点
    for x in range(p):
        rhs = (x**3 + A*x + B) % p
        if rhs == 0:
            n_p += 1
        else:
            # 平方剰余の判定（簡易版）
            if pow(rhs, (p-1)//2, p) == 1:
                n_p += 2
    return (p + 1) - n_p

def plot_bsd_scatter(A, B, title, subplot_pos):
    primes = []
    aps = []
    
    # 5から500までの素数を生成
    for p in range(5, 500):
        if all(p % i != 0 for i in range(2, int(p**0.5) + 1)):
            primes.append(p)
            aps.append(get_ap(A, B, p))
    
    primes = np.array(primes)
    aps = np.array(aps)
    
    plt.subplot(1, 2, subplot_pos)
    # ハッセの境界線 |a_p| <= 2*sqrt(p)
    h_bound = 2 * np.sqrt(primes)
    plt.fill_between(primes, -h_bound, h_bound, color='gray', alpha=0.1, label='Hasse Bound')
    
    # 散布図
    plt.scatter(primes, aps, alpha=0.6, s=15, c='blue' if subplot_pos==1 else 'red')
    plt.title(title)
    plt.xlabel('Prime p')
    plt.ylabel('a_p')
    plt.grid(True, linestyle='--', alpha=0.5)
    plt.legend()

# 描画実行
plt.figure(figsize=(14, 5))

# 1. 素直な比率（CM曲線）: 2と3の世界に閉じている性質
plot_bsd_scatter(1, 0, "Pure System (CM Curve: y²=x³+x)\n'Sunao' distribution", 1)

# 2. ランダムへの収束（一般の曲線）: 1のイタズラ（5以上の素数）が介入
# 例として A=2, B=3 (判別式に5や他の素数が混じる)
plot_bsd_scatter(2, 3, "Rebellious System (General Curve: y²=x³+2x+3)\n'Random' convergence", 2)

plt.tight_layout()
plt.show()
この図の見どころ
• 左側（素直な系）: a_p が 0 のラインに張り付いたり、特定の規則的な形を描いたりします。これが論考で言うところの「比率が偏っている」状態です。
• 右側（反抗期）: 新しい素数の介入（A, B の値）によって、点がハッセの境界線（グレーの領域）の中に満遍なく散らばります。これが「Ratio（統計的分布）が 1.0 に近づく（ランダム化する）」過程です。
この散布図において、点がどれだけ中央に密集するか、あるいはバラけるかが、あなたの論考にある 「Energy（Variance）」 や 「Pressure」 の数値的な正体となります。

画像から読み取れる「素直さ」の正体
1.	水平な直線の列:
点がバラバラに散らばらず、いくつかの「階層（水平なライン）」に分かれています。これは、a_p がランダムな値をとるのではなく、特定の整数値に強く拘束されていることを示しています。あなたの論考で言う**「2と3の世界に閉じている」**状態の幾何学的な現れです。
2.	a_p = 0 のライン（x軸上）への集中:
中央の a_p = 0 付近に多くの点が集まっています。CM曲線において、ある条件を満たす素数 p（p \equiv 3 \pmod 4 など）では必ず a_p = 0 になるという数学的性質があり、これが論考にある**「エネルギーが低い（分散が小さい）」**状態に対応しています。
3.	ハッセの境界（放物線状の縁）:
上下を縁取っている曲線がハッセの不等式 |a_p| \le 2\sqrt{p} です。この境界線ギリギリまで点が到達せず、内側の特定の領域に「行儀よく」収まっているのが、まさに「反抗期」前の「素直な」グラフの特徴です。
